# 定制手办独立站 - 技术方案

## 一、项目架构

```

│  │  - 首页 / 商品展示 / 3D预览 / 购物车 / 结账              │   │
│  │  - React Three Fiber 实现 3D 渲染展示                   │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐

│  ┌──────────────────┐  ┌──────────────────┐                   │
│  │  3D 渲染服务      │  │  Shopify 集成    │                   │

│  └──────────────────┘  └──────────────────┘                   │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼

└─────────────────────────────────────────────────────────────────┘
```

## 二、技术栈选型

### 前端
| 技术 | 选型 | 理由 |
|------|------|------|
| 框架 | **Next.js 14** (App Router) | SEO 好、SSR/SSG 支持、生态成熟 |
| UI 组件 | **Tailwind CSS + shadcn/ui** | 快速开发、现代化设计 |
| 图片展示 | **通用图片组件** | 展示AI生成的图片 |
| 状态管理 | **Zustand** | 轻量、适合电商场景 |
| 表单处理 | **React Hook Form** | 性能好、易于验证 |

### 后端
| 技术 | 选型 | 理由 |
|------|------|------|
| API | **Next.js API Routes** | 统一代码库，无需额外服务 |
| AI 生图 | **待集成商业API** (如 Replicate、Stability AI) | 用户上传图片生成3D风格渲染图 |
| Shopify 集成 | **@shopify/shopify-api** + **GraphQL Storefront API** | 官方推荐方案 |

### 部署
| 技术 | 选型 | 理由 |
|------|------|------|
| 托管 | **Vercel** | Next.js 官方支持，免费额度够用 |
| CDN | Vercel Global Edge Network | 面向海外速度快 |

## 三、功能模块

### 1. 首页 (/)
- 品牌展示
- 主打定制手办展示
- CTA 引导用户开始定制

### 2. 定制流程 (/customize)
- Step 1: 用户上传原始图片
- Step 2: 调用 AI 生图 API 生成 3D 风格渲染图
- Step 3: 用户预览确认生成的图片
- Step 4: 选择规格（尺寸、数量等）加入购物车

### 3. 商品展示 (/products)
- 展示不同类型/尺寸的手办
- 价格信息

### 4. 购物车 (/cart)
- 添加/删除商品
- 修改数量

### 5. 结账 (/checkout)
- Shopify Checkout 重定向
- 支持信用卡、PayPal 等支付方式

### 6. 订单管理（后期）
- 线下流程同步（物流信息等）

## 四、AI 生图模块

### 初期方案（测试用）
- 简单的图片处理 API
- 能返回处理后的图片即可测试流程

### 正式方案
- 对接商业 AI 生图 API（如 Replicate、Stability AI）
- 用户上传 2D 图片 → AI 生成 3D 风格渲染图

## 五、Shopify 集成要点

### 需要你提供的 Shopify 信息
1. **Shopify 店铺域名**（如 xxx.myshopify.com）
2. **Storefront API 访问令牌**（需要我在 Shopify 后台生成）
3. **商品创建**：基础商品（不同尺寸的手办）

### 集成方式
- 使用 **Storefront API** 的 GraphQL 接口
- 创建自定义商品（Custom Product）用于定制订单
- 结账时跳转到 Shopify Checkout 页面完成支付

## 五、3D 渲染模块（中期）

### 初期测试方案
- 简单的图片处理 API（Node.js + Canvas 或服务端渲染）
- 能返回处理后的图片即可

### 正式方案（后续）
- 对接 AI 3D 生成 API（如 Replicate、Stability AI）
- 或用户自研的 3D 渲染服务
